# Paysage API

## Developpement
### Install
`npm install`

### Start the app in dev
`npm run start`

### Run tests
* Run all tests

`npm test`

* Run only one tests file

`npm test my_file.test.js`

* Run test into a Docker container

`docker-compose -f docker-compose-test.yml up --build --exit-code-from app`
